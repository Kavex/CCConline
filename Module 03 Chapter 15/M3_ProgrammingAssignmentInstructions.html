Chapter 15 Programming Assignment (Version 1)

Overview

This assignment relates to content from Chapter 15 of the eText.

Instructions

  1. Review the General Programming Assignment instructions.
  2. A palindrome is a string that reads the same both forward and backward. For example, the string "madam" is a palindrome.
    1. Write a program that uses a recursive function to check whether a string is a palindrome.
    2. Your program must contain a value-returning recursive function that returns true if the string is a palindrome and false otherwise.
    3. Do not use any global variables; use the appropriate parameters.
